# Personal Finance Tracker

A command-line finance tracking tool built in Python with AI-powered financial analysis using the Anthropic API.

## What it does
- Add and track regular income (weekly/monthly/daily) with automatic monthly normalisation
- Add and track one-off income
- Add and track regular expenses by category
- Add and track one-off expenses
- View a detailed financial summary with totals and monthly balance
- AI-powered financial advisor summary using Claude (Anthropic API)

## Tech used
- Python 3.12
- Anthropic API (claude-sonnet-4-6)
- python-dotenv (secure API key handling)
- JSON (local data persistence)

## Known limitations
- Data is only saved when exiting via option 6 (planned: auto-save)
