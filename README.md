# Personal Finance Tracker

A command-line finance tracking tool built in Python with AI-powered spending insights using the Anthropic API.

## What it does
- Track regular and one-off income
- Track regular and one-off expenses by category
- View spending summary with category breakdowns and percentages
- AI suggestions on where to cut costs

## Tech used
- Python 3.12
- Anthropic API
- python-dotenv (secure API key handling)

First project in my AI builder journey — learning Python and AI API integration by building real tools.
